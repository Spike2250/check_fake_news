## ML-project to identify fake news with simple GUI
preprocessing - TfidfVectorizer

model - PassiveAggressiveClassifier

dataset - "data/fake_news.csv"
### Installation
```ch
poetry install
```
### Usage
```ch
poetry run start-app
```
